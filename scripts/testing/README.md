# Testing Scripts

Cross-browser and smoke-test scripts for CI/CD.

## Files

- `audit-demo.js` — Audit demo script
- `browserstack-smoke-test.js` — BrowserStack smoke test
- `lambdatest-smoke-test.js` — LambdaTest smoke test
- `testingbot-smoke-test.js` — TestingBot smoke test

## Usage

These scripts are typically run from GitHub Actions. See `.github/workflows/` for the exact integration.
