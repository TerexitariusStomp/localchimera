# Debug Scripts

One-off debugging and inspection scripts for Localchimera contracts and deployments.

## Files

- `debug-deploy*.mjs|ts` — Debug deployment transactions and receipts
- `debug-tx*.mjs` — Debug general transaction issues
- `test-original-args.ts` — Test original constructor arguments

## Usage

```bash
npx tsx scripts/debug/debug-deploy.ts
npx tsx scripts/debug/debug-tx.mjs
```
