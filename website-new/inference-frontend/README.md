# website-new/inference-frontend

Next-generation React frontend for the Chimera encrypted inference marketplace.

## Development

```bash
cd website-new/inference-frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build output is served from `website-new/inference/`.
